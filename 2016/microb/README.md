## Bacillus
[バシラス属](https://ja.wikipedia.org/wiki/バシラス属)

この属には、[枯草菌](https://ja.wikipedia.org/wiki/枯草菌)(B. subtilis)や納豆菌、病原菌の[炭疽菌](https://ja.wikipedia.org/wiki/炭疽菌)(B. anthracis)やセレウス菌(B. cereus)などの種も含まれる。
### 参考
- [納豆菌](https://ja.wikipedia.org/wiki/納豆菌)（なっとうきん、学名Bacillus subtilis var. natto）は、枯草菌の一種である。
- [アメリカ炭疽菌事件 - Wikipedia](https://ja.wikipedia.org/wiki/アメリカ炭疽菌事件)
- 論文・データベース
 - [Genome Announc. 2013 Oct 3; Whole-Genome Sequences of 94 Environmental Isolates of Bacillus cereus Sensu Lato.](http://www.ncbi.nlm.nih.gov/pubmed/24092776)
 - [Phylogenetic Profile for COG5013, COG1140](http://www.microbesonline.org/cgi-bin/phyloprofile.cgi?group=COG5013&group=COG1140&download=0&show=astree&group=&taxId=g1386)

- [私たちの体にいる細菌と健康 腸内は新種の細菌がいっぱい？](http://naohiroinohara.blog.fc2.com/blog-entry-17.html)
もっともひどいのは、セレウス菌Bacillus cereusと炭疽菌Bacillus anthracisの関係です。両者は「同じ種」なのに、毒素遺伝子を載せたプラスミドを持って炭疽を起こす菌は炭疽菌と呼ばれています。それでは炭疽菌として単離されてプラスミドをなくしたものはなんと呼ぶのか、悩みます。日本の法律では、プラスミドをなくしたものでも炭疽菌は炭疽菌ですが、科学的には単なるセレウス菌ですものね。
