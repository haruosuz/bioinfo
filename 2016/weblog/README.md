## [カルバペネム耐性腸内細菌](https://ja.wikipedia.org/wiki/カルバペネム耐性腸内細菌)（Carbapenem-resistant enterobacteriaceae、CRE）
薬剤耐性菌の一種。CREが産生するカルバペネマーゼ（β-ラクタム系抗生物質を加水分解する酵素のこと）3つのグループ：  
- IMP型、VIM型、NDM型など、メタロ-β-ラクタマーゼ（MBL）のグループ
- KPC (Klebsiella pneumoniae Carbapenemase) 型
- OXA-48型などの新型カルバペネマーゼ

## [ピロリ菌](https://ja.wikipedia.org/wiki/ヘリコバクター・ピロリ)
ヘリコバクター・ピロリ (Helicobacter pylori) とは、ヒトなどの胃に生息するらせん型のグラム陰性微好気性細菌である。単にピロリ菌と呼ばれることも多い。1997年にはゲノム解読が完了した[PMID: 9252185](https://www.ncbi.nlm.nih.gov/pubmed/9252185)。病原因子[CagA (cytotoxin-associated gene A)](https://ja.wikipedia.org/wiki/ヘリコバクター・ピロリ#CagA)を持つピロリ菌株に感染した場合、持たない菌株の感染よりも消化性潰瘍や胃癌になるリスクが高い。

## [リケッチア目](https://ja.wikipedia.org/wiki/リケッチア目)(Rickettsiales)
αプロテオバクテリアのリケッチア目に属する細菌の多くは他の生物の細胞内でのみ生存可能である。リケッチアのような病原体が含まれる一方、細胞内共生説でミトコンドリアの起源となった細菌もここに由来すると考えられる。

## [ボルバキア](https://ja.wikipedia.org/wiki/ボルバキア)(Wolbachia)
ボルバキアは、リケッチア目のアナプラズマ科に属し、節足動物の体内に生息する共生細菌の一種で、昆虫宿主の生殖システムに影響を与えることが知られている。オス殺し、メス化（性転換）、単為生殖、細胞質不和合など。

http://rest.g-language.org/help/gcskew

         'protein_id' => 'YP_095241.1',
